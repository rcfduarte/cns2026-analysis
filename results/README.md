# Results — figures & tables

Publication figures (PNG + SVG) and aggregate tables, regenerated by
`python scripts/run_all_figures.py`. These are **aggregate statistics only** — no raw
abstract text. Narrative for each is in [`../docs/RESULTS.md`](../docs/RESULTS.md).

## Figures (`figures/`)

| File | What it shows |
|------|---------------|
| `fig1_aiml_rise` | The AI/ML wave at CNS (2007–2025); LLMs appear from 2023 |
| `fig2_theme_heatmap` | All keywords, row-normalised intensity, sorted by era (rising vs enduring vs fading) |
| `fig3_cosyne_cns_divergence` | "Two communities, two vocabularies" — CNS-vs-CoSyNe keyword gap |
| `fig4_aiml_uptake` | AI/ML adoption over time: CoSyNe vs CNS |
| `fig5_topics_over_time` | 10 largest BERTopic topics, volume-normalised share of abstracts |
| `fig6_internationalization` | Distinct countries + geographic diversity (entropy) per meeting |
| `fig7_countries` | Author affiliations by country over time (top 10) |
| `fig8_robustness` | Title-only vs full-abstract keyword-trajectory correlation (titles are lossy) |

## Tables (`tables/`)

| File | Contents |
|------|----------|
| `keyword_frequencies.csv` | keyword × year × per-10k-words frequency (full-text tier) |
| `cosyne_cns_contrast.csv` | per-keyword mean frequency at each conference + gap |
| `network_summary.csv` | author / collaboration / abstract counts |

Topic catalogue (`topic_catalogue.csv`) and per-doc topic assignments are produced by
`scripts/run_topics.py` into the gitignored `outputs/topics/` (they derive from abstract
bodies); re-run locally to regenerate.
